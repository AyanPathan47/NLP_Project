Project Overview

This project implements an Automatic Extractive Notes Summarization system using Natural Language Processing (NLP) techniques. It was developed as an academic project for MSc Computer Science evaluation.

The core idea: given a long piece of academic text, the system identifies the most important sentences using statistical analysis (TF-IDF) and presents them as a concise summary ‚Äî without generating any new text or calling external AI services.

Academic Context

Text summarization is one of the core tasks in NLP research. It falls into two broad categories:

Extractive Summarization ‚úì

Selects the most important sentences from the original document. No new text is generated. Uses statistical methods like TF-IDF, TextRank, or LSA.

Used in this project

Abstractive Summarization

Generates new sentences that paraphrase the original content. Requires deep learning models (BART, T5, GPT) and large training datasets.
